Anaconda

If you have Anaconda installed on your computer, use that for the sessions for the next two days. Follow the instructions in the first tab titled ‘Anaconda’.

Alternative 1: Kaggle

If you don’t have Anaconda installed, the next preferable option is Kaggle. Follow the instructions in the second tab titled ‘Kaggle’.

Alternative 2: Jupyter online

If you not able to access Anaconda or Kaggle, your final option is Jupyter online. Follow the instructions in the third tab titled ‘Jupyter online’.

Instructions

  • Anaconda
  • Kaggle
  • Jupyter Online
  1. Open Anaconda Prompt

  1. Once the Anaconda Prompt is open, navigate to the desired folder using the cd command.

  1. Once you are in that specific folder type jupyter notebook and press the enter key.

  1. The Jupyter server will start. You should see some server logs printed. Do not close this window, minimise it.

  1. Shortly after, a browser window will open, showing the files and folders located in the folder where you started the Jupyter server (in my case, this folder is D:/Induction Python).

  1. At the top-right of this file view, click New and select Python 3 from the dropdown. This will open a new Jupyter Notebook and you’re ready to start!

  1. Go to Kaggle.com and create an account if you don’t already have one.

  2. Log into you Kaggle account.

  3. In the left menu, goto Create > New Dataset

  1. In your file explorer, click and drag the csv file into the highlighted area.

  1. Name the dataset Ames_dataset and click Create.

  1. When you receive the message saying “Success!”, click the X in the top-left of that pane.

  2. In the left menu, goto Create > New Notebook

  1. In the right-hand Data panel of the Notebook goto Add Data

  2. In the filters below the search bar click Your Datasets

  1. You can now see your dataset Ames_dataset. Click the + to the right of the dataset name.

  2. Click the X in the top-right corner of the Add Data panel to close it.

  3. In the right-hand Notebook panel, click the little arrow next to the name of the dataset.

  4. You’ll now see the name of the csv file we uploaded. To the right of the file name there is a little “copy” icon. You will need this to copy the file path so that you can use the data in these sessions.
